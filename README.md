# Diet-o Frontend

A React + Vite frontend project built with TypeScript and Tailwind CSS. This repository includes a landing page style app with animated hero content and floating decorative shapes.

## Features

- React 19 + TypeScript
- Vite development build system
- Tailwind CSS styling
- Framer Motion animations
- Hero section with floating shapes and responsive layout

## Getting Started

### Prerequisites

- Node.js 18+ or newer
- npm

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open the local URL shown in the terminal to view the app.

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Project Structure

- `src/` - application source files
- `src/components/` - reusable React components
- `src/styles/` - component-specific CSS modules
- `public/` - static assets
- `pages/` - page entry files
- `tsconfig.json` - TypeScript configuration
- `vite.config.ts` - Vite configuration

## Notes

- The project currently uses the package name `react-vite-tailwind` in `package.json`.
- Add or replace assets in `public/images/` as needed.
